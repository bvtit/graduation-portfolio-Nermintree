Project Description

A few peers and I were assigned a customers Ipad that didn't seem to charge for more than a few seconds. The first method was using different outlets with different cables. After that didn't help the issue, we deduced that it must be something internal. So we took the casing, heated one area and was able to pop the screen off revealing the inside. We started disconnecting screws and looking to see if the port was plugged in. After confirming it was we tested a few more issues we figured the only way to do is to order a new screen and charging port. Currently we await the order to come in.
